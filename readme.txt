Generate beautiful steps for your page.

== Description ==

With this plugin, you can generate them simple but beautiful steps / guild box for you page.
Just fill the blank on the plugin settings, then see the result on the client page.

== Installation ==

1. Direct install or upload `beautiful-steps` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Fill the required fields on the plugin settings, insert a shortcode to any page, then enjoy.
